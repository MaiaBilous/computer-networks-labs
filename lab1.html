Лабораторна робота №1

Завдання

Розробити програму мовою Python, яка реалізує команду ping для перевірки доступності мережевого вузла.

Принцип роботи програми

Користувач вводить адресу хоста (IP або домен) та кількість пакетів. Програма за допомогою бібліотеки pythonping надсилає ICMP-запити до вказаного сервера.

Після завершення перевірки на екран виводиться інформація про відповіді від хоста та статистика часу затримки.

Код програми

index.py


from pythonping import ping

def main():
    host = input("Введіть адресу хоста: ")
    count = int(input("Кількість пакетів: "))

    print(f"Ping {host} ({count} пакетів)")
    result = ping(host, count=count)

    print(result)

if __name__ == "__main__":
    main()

requirements.txt


pythonping==1.1.4
